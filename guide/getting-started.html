🚀 快速开始 ​

获取KLineChart ​

KLineChart 支持多种下载方式，你可以通过npm或yarn等包管理工具，也可以通过CDN获取。

使用npm ​

bash
npm install klinecharts --save

使用yarn ​

bash
yarn add klinecharts

使用CDN ​

可以使用jsDelivr，unpkg或者其它的，推荐从jsDelivr引用。

bash
<script type="text/javascript" src="https://cdn.jsdelivr.net/npm/klinecharts/dist/klinecharts.min.js"></script>

创建第一个图表 ​

在npm或yarn等包管理工具的项目中创建 ​
